FAQ

よくあるご質問

初期費用は本当に無料ですか？ +
はい、AURUM LUXのシステム導入費用・利用料は一切かかりません。指定ブローカー「Doo Prime」にて口座開設いただくことで無料でご利用いただけます。取引に伴うスプレッドは通常通り発生します。
いくらから始められますか？ +
推奨は100万円以上ですが、70万円以下の方はセント口座を開設することで、実質10万円程度の少額からでも安全に稼働可能です。セント口座はリスクを1/100に抑えた練習環境として最適です。
運用が失敗したらどうなりますか？ +
Doo Prime採用のゼロカットシステムにより、入金額以上の損失（追証）は一切発生しません。最大損失は口座への入金額までと保証されています。また各システムには動的DD抑制機能が内蔵されています。
EAの稼働を止めるタイミングは？ +
重要度★★★の経済指標発表日（FOMC, CPI, 雇用統計など）はEA停止を推奨しています。メニューバーのEA稼働危険日カレンダーで今週の危険日を一目で確認できます。
サポート体制は？ +
導入後は運用テンプレート配布・経済指標対応のガイダンス・稼働状況チェックリストを提供します。不明点はInstagramのDMから随時ご相談いただけます。

THE STORY

ニューヨーク大手ブローカーでの 8年間、アルゴリズム取引の運用とファンドマネージャー業務に従事。その後シドニーに拠点を移し、台湾・日本・インドの計 11社と連携しながら、金融とテクノロジーを横断する事業開発に取り組んできました。

その過程で繰り返し見てきたのが、EA詐欺被害の相談です。日本とオーストラリアの弁護士事務所と提携して実際の事件解決を担う中で、被害者の多くが「まともなロジックを持った EA に出会えれば、そもそも被害に遭わなかった」という事実に直面しました。

そこで実感したのは「個人投資家が機関投資家に勝てない根本原因は、情報・速度・資金力の格差ではなく、"アルゴリズム思考" の不在」ということ。

オークション理論・Volume Profile・Order Flow という、本来プロが使う分析フレームを、日々の小さな取引に組み込める形に再設計したのが AURUM LUX です。

「勝てる仕組みは、一部の人の特権であってはならない」——これが AURUM を開発した理由であり、無料で提供し続ける理由です。

お名前.com VPS の契約

所要時間: 約15分 / 月額1,000円〜

AURUM EA は VPS (仮想専用サーバー) 上で24時間稼働させます。
推奨: お名前.com デスクトップクラウド for MT4/MT5。MT5 専用設計で、初期設定が最も簡単です。

1-1

公式サイトにアクセス

お名前.com のVPSサービス公式ページから「FX専用デスクトップクラウド」を選択。

お名前.com デスクトップクラウド (FX専用) を開く ›
1-2

プラン選択 (推奨)

  • プラン1.5GB プラン
  • 月額1,860円〜
  • CPU仮想2コア
  • ディスクSSD 50GB
  • OSWindows Server 2022

※ 3システム並列運用なら 2GB プラン以上を推奨。

1-3

申し込みフロー

  1. 「お申し込み」ボタンをクリック
  2. お名前ID (お名前.comの会員ID) でログイン (未登録なら新規登録)
  3. プラン・契約期間を選択 (1ヶ月・3ヶ月・6ヶ月・12ヶ月)
  4. 支払い方法を選択 (クレカ / 銀行振込 / コンビニ払い)
  5. 申込確定 → 数分以内に「設定完了通知メール」受信
📩 重要: メールに記載される「VPS接続情報」(IPアドレス・ユーザー名・初期パスワード)は必ず保存してください。EA設置に必須です。
02

VPS への接続

所要時間: 約5分 / 全OS対応

契約完了後、自分のPC・スマホからVPSのデスクトップに接続します。
OSによって使うアプリが異なります。

Windows

標準「リモートデスクトップ接続」

  1. スタートメニュー → 「リモートデスクトップ接続」を起動
  2. コンピューター欄に VPSのIPアドレスを入力
  3. ユーザー名・パスワードを入力 → 接続
  4. VPSのWindowsデスクトップが表示される
Mac / iOS

「Microsoft Remote Desktop」

  1. App Store / Mac App Store から無料DL
  2. 「Add PC」 → IPアドレスを入力
  3. User account を追加 → ユーザー名・パスワード
  4. PCをタップ → VPS接続完了
Android

「Microsoft Remote Desktop」

  1. Google Play から無料DL
  2. 右上「+」 → 「PC」を追加
  3. PC name に IPアドレスを入力
  4. User account: ユーザー名 / パスワード
  5. 追加された PC アイコンをタップ → VPS接続
💡 Android Tip: 横画面推奨、Bluetoothキーボード接続でMT5操作が快適に。
⚠️ セキュリティ: 初回ログイン後、必ずパスワードを強力なものに変更してください (12文字以上・英数記号混合推奨)。
03

MT5 + EA ファイルの設置

所要時間: 約10分

VPS のデスクトップ上で MT5 (MetaTrader 5) をインストール、AURUM EA ファイルを配置します。

3-1

Doo Prime MT5 のダウンロード

  1. VPS内のブラウザ (Edge / Chrome) で Doo Prime公式サイトを開く
  2. 「ダウンロード」 → 「MT5 for Windows」
  3. インストーラーを実行 → デフォルト設定でインストール
  4. MT5起動 → ログイン: Doo Prime ID + パスワード + サーバー選択
3-2

EA ファイルの配置

当方から 「2026」または「rux」と記載されたEAファイルを配布します。

  1. 受け取った .ex5 ファイル(2026_*.ex5 または rux_*.ex5) をVPSにダウンロード
  2. MT5メニュー: ファイル → データフォルダを開く
  3. 開いたフォルダ内の MQL5 → Experts へ .ex5 ファイルをコピー
  4. MT5を再起動 (またはナビゲータパネルで右クリック → 更新)
  5. 左パネル「ナビゲータ」 → 「エキスパートアドバイザ」に 2026_*.ex5 / rux_*.ex5 が表示される
04

EA の起動

所要時間: 約5分

4-1

自動売買の有効化

  1. MT5上部ツールバーの 「自動売買」ボタンをクリック (緑になればON)
  2. メニュー: ツール → オプション → エキスパートアドバイザ
  3. 以下にチェック:
  4. 自動売買を許可する
  5. DLLの使用を許可する
  6. WebRequestを許可するURLリスト
4-2

EAをチャートに適用

  1. 運用通貨ペアのチャートを開く (XAUUSD / BTCUSD)
  2. 時間足を M1 (1分足) に設定 ※全システム共通推奨
  3. ナビゲータから 2026_*.ex5 または rux_*.ex5 をチャートにドラッグ&ドロップ
  4. パラメータ設定ダイアログが表示 → 「OK」で起動
  5. チャート右上にEA名 + 以下のマークが表示されたら稼働中:
  6. 😊 ニコちゃんマーク = MT4 で稼働中
  7. 🎩 帽子マーク = MT5 で稼働中
💡 推奨時間足: 全システム共通で M1 (1分足) でのご利用を推奨しています。短期決済で安定した収益を狙えます。
05

パラメータの開き方

配布ファイルからパラメータ設定画面を開く方法

配布した 「2026」または「rux」と記載されたファイル からパラメータ設定画面を開く手順です。

5-1

EA起動時にパラメータダイアログを開く

  1. MT5左パネル「ナビゲータ」 → 「エキスパートアドバイザ」を展開
  2. 配布ファイル 2026_*.ex5 または rux_*.ex5 を見つける
  3. ファイルを右クリック → 「チャートに表示」
    または、ファイルをチャート上にドラッグ&ドロップ
  4. パラメータ設定ダイアログが自動的に開く
  5. 「パラメーター」タブを開く → 全設定項目が一覧表示される
  6. 初期値のまま「OK」で稼働可能
5-2

稼働中のEAパラメータを変更する

  1. 稼働中のチャートを右クリック
  2. 「エキスパートアドバイザ」 → 「設定」をクリック
  3. パラメータ設定ダイアログが開く → 値を変更 → 「OK」
  4. 変更が即時反映される
💡 ショートカット: チャート上で F7キー を押しても同じダイアログが開きます。
5-3

パラメータの保存と復元

  1. パラメータダイアログ右下「保存」ボタン → 設定を .set ファイルとして保存
  2. 「読込」ボタン → 保存した .set ファイルから設定を復元
  3. 当方推奨設定の .set ファイルも併せて配布いたします
